# Hello, world

![task 00](../../badges/tasks/task_00.svg)

Задача на разогрев: научиться запускать go test и писать самую простую функцию, которую проверяет тест.

Что нужно сделать  
1) В проекте уже есть тест Test_greet. Он вызывает функцию greet() без аргументов.  
2) Твоя задача — реализовать функцию greet() так, чтобы тест проходил.

Требование к функции  
- Пакет: package main  
- Сигнатура: func greet() string  
- Возвращаемое значение: строго строка "Hello, World!" (обрати внимание на регистр букв, пробел после запятой и !)

Важно  
- Ничего не печатай в stdout (fmt.Println не нужен). Тест проверяет именно _возвращаемую_ строку.  
- Не добавляй перевод строки \n и лишние пробелы.  
- Название функции должно совпадать с тестом: greet (не hello).
