# Greeting

![task 01](../../badges/tasks/task_01.svg)

Задание для практики: написать простую функцию и научиться запускать `go test`.

---

Задача

Реализуйте функцию `greet`, которая возвращает приветствие по имени.

Сигнатура:

    func greet(name string) string

Поведение:

- если `name` — пустая строка, функция должна вернуть:
  `Hello, World!`

- если `name` не пустая, функция должна вернуть:
  `Hello, <name>!`

Важно: формат строки должен совпадать *точно* (заглавные буквы, запятая, пробел, восклицательный знак).

Примеры:

- greet("")      -> "Hello, World!"
- greet("Alice") -> "Hello, Alice!"
